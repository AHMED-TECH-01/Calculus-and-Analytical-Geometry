# Calculus and Analytical Geometry Course Reference Library

This repository functions as a structured document repository for the Calculus and Analytical Geometry course curriculum. It hosts academic PDFs, lecture note packages, coordinate system graphs, and step-by-step problem-solving manuals covering single-variable calculus, infinite series, and vector spaces.

## Course Content and Topic Coverage

The archived PDF files correspond directly to standard mathematical phases, leading from primary limits to multi-dimensional coordinate fields.

### Unit 1: Limits, Continuity, and Differentiation
* Limits of Functions: One-sided limits, infinite limits, and algebraic evaluation techniques.
* Continuity: Intermediate Value Theorem and extreme value parameters.
* The Derivative: Chain rule, implicit differentiation, and higher-order derivatives.
* Applications of Derivatives: Related rates, optimization problems, and curve sketching.

### Unit 2: Integral Calculus
* Riemann Sums: Definite integrals and the Fundamental Theorem of Calculus.
* Integration Techniques: Substitution, integration by parts, trigonometric integrals, and partial fractions.
* Applications of Integrals: Areas between curves, volumes of solids of revolution, and arc length.

### Unit 3: Sequences and Infinite Series
* Convergence Criteria: Integral test, comparison tests, ratio test, and root test.
* Power Series: Radius of convergence, Taylor series, and Maclaurin expansions.

### Unit 4: Analytical Geometry and Vector Spaces
* Coordinate Geometry: Conic sections (parabolas, ellipses, hyperbolas) and rotation of axes.
* 3D Vector Systems: Dot product, cross product, lines and planes in space, and cylindrical/spherical coordinates.

## Access and Usage

### Prerequisites
A modern web browser or dedicated document processor (such as Adobe Acrobat Reader) is necessary to access the vector graphics and text within these PDFs.

### Local Download Instructions
To pull down the complete geometry and calculus document suite:
1. Clone the repository to your desktop machine:
   ```bash
   git clone https://github.com
   ```
2. Access the target unit folder:
   ```bash
   cd calculus-geometry-course/materials
   ```
